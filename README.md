# Kasey Burton Artist HQ

This repo is the home base for Kasey Burton's music career systems: brand, songs, releases, content, budget, live opportunities, website planning, and assistant context.

If you are new to Codex, start here:

1. Open `00-start-here/how-to-use-this-repo.md`.
2. Open `08-weekly-planning/this-week.md`.
3. Ask Codex to walk through the weekly reset with you.

The goal is simple: keep the creative life organized enough that it can grow.

## Main Rooms

- `01-brand`: Kasey & The Sunshine Club, public boundaries, story, slogans, and sound.
- `02-music-catalog`: songs, lyrics, song stories, audio notes, and release readiness.
- `03-release-strategy`: rollout plans, project arcs, and release checklists.
- `04-social-media`: post ideas, captions, filming days, and reward systems.
- `05-live-shows-and-opportunities`: gig leads, booking outreach, and paid opportunities.
- `06-budget-and-business`: income, expenses, budget rules, and merch ideas.
- `07-website`: website plans, SEO ideas, and future Vercel build.
- `08-weekly-planning`: weekly reset, habits, content calendar, and focus.
- `09-assistant-system`: context files that help Codex act like Kasey's assistant.

## Local Dashboard

The simple browser dashboard lives in `app/`.

To run it, use:

```bash
node server.mjs
```

Then open:

```text
http://localhost:4173
```
